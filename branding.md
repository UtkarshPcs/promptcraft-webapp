Typography
Role - Font - Why

Display / Hero - Sora - Geometric, modern, slightly distinct — not overused

Body / UIDM Sans - Readable, neutral, clean

Mono / Code - JetBrains MonoIf you show any code snippets


Type scale: Large, confident headings (56–72px hero) with compact, tight line-height. Body at 16–18px with generous line-height (1.6).

Color Palette
Background:   #0A0A0A  ← near-black, not pure black
Surface:      #111111  ← card/panel layer
Border:       #222222  ← subtle dividers
Muted text:   #666666
Body text:    #CCCCCC
Headings:     #F5F5F5  ← near-white
Accent:       #E8E8E8  ← bright white-gray for CTAs
One optional accent pop — a single restrained color like #7EFFC5 (mint) or #4DFFB4 (a cool green) for hover states and CTAs only. This is optional but prevents the UI from feeling lifeless.

Design Principles

Generous whitespace — let sections breathe
Borders over shadows — use 1px borders instead of box shadows
Monochrome icons — Lucide or Phosphor icon sets
Micro-animations — subtle fade-ins on scroll, hover underlines, smooth transitions (200ms ease)
Asymmetric layouts — break the grid occasionally in hero sections
